# 📰 Fake News Detection in Tamil using NLP & Deep Learning

A Natural Language Processing (NLP) and Deep Learning project that automatically classifies Tamil news headlines as **Fake** or **Real**. The project leverages language-specific text preprocessing, feature extraction, and classification techniques to help combat the spread of misinformation in Tamil digital media.

---

## 📖 Project Overview

Fake news has become a major challenge in today's digital world, influencing public opinion and spreading misinformation rapidly. While many fake news detection systems focus on English, this project addresses the need for reliable detection in the Tamil language.

The system preprocesses Tamil text, extracts meaningful features, trains classification models, and predicts whether a given news headline is genuine or fake.

---

## ✨ Key Features

- 📝 Tamil text preprocessing and cleaning
- 🔍 Feature extraction using TF-IDF
- 🤖 Machine Learning and Deep Learning-based classification
- 📊 Model evaluation using standard performance metrics
- ✅ Prediction of Fake and Real Tamil news headlines
- 🌐 Streamlit-based interactive user interface
